## Canned Laughter — The Chimera's Infinite Jest
**Canned Laughter** is a mod originally created out of a simple query; that being "*Can I really make a mod on my own?*" The answer turned out to be no, but I got pretty close.

It's also a mod about repurposing, and recontextualizing, inspired loosely by a lot of the games I found myself playing over the last few years, that being Roblox games with extraordinarily deep secrets. Here, you may find yourself delving into semi-obscure mechanics, or finding new ones where there weren't any, all without making it too complicated.

This mod was fully scripted and primarily illustrated by [ProtoAno](https://github.com/ProtoAnoWasTaken) (ProtoAnoWasTaken), with some guest art by [Faow](https://github.com/faowfanatic-cell) (Faowbot), to which I owe a great favor to over in our partner project, [Aspirant](https://github.com/ProtoAnoWasTaken/Aspirant).

Conceptualization for mod content comes from both existing and unused work in vanilla Balatro, such as the ever-iconic LumpyTouch unused joker and various items from the demo version, as well as games that Balatro has crossed over into, if they have any content related to the game to speak of.

## FAQ
> **Does this mod have any dependencies?**
>> This mod currently requires [Steamodded](https://github.com/Steamodded/smods/), and probably will forever.

> **Does it have any compatibilities with other mods?**
>> The most compat it currently has is the general Food Joker pool that a lot of mods, primarily Cryptid and its branches, have.
>> There are also a few outgoing compats that *your* mod can add: that being Banana Jokers as a category, as well as its extinction cycle system, adding more Trials and Trial reps, and some compats for a special Boss Blind.

> **How much content is currently in the mod?**
>> As of v1.0, Canned Laughter adds 61 jokers, 28 boss blinds, 8 decks, 3 seals and editions (with sealing compat for Jokers), 10 Tags, 6 tarot, and 7 spectrals. It also features 14 achievements **necessary** for the unlocking of certain jokers.
>> It also features an entirely-new **Barter** system, for if you're feeling real greedy for whatever's in the booster pack, with 28 unique trials within to conquer. 

> **Will you add more to the mod?**
>> As it stands? The most I can see myself doing with the current set-up is making more compat updates and bug-fixes. If I get struck with enough inspiration, though, perhaps we'll see more real content.

> **Why isn't the mod loaded?**
>> You might not have extracted it correctly. If there's a Canned Laughter folder *inside of* the Canned Laughter zip archive, you need to make sure it's only one deep.
